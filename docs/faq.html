Common questions about arTui

What is arTui?

arTui is a terminal user interface to browse recent arXiv papers, filter results, and build a personal reading library.

Is arTui free?

Yes. It is open source and released under the MIT license.

Can I customize categories and filters?

Yes. You can define your arXiv categories and optional keyword filters in config.yaml.

Does it support notes and tags?

Yes. Saved papers can be organized with custom tags and markdown notes for long-term tracking.

Where can I report bugs or feature requests?

Use the GitHub repository issues page: github.com/fjonasALICE/arTui/issues

arTui helps researchers stay current without leaving the terminal.
